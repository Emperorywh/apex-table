<h1>ApexTable</h1>

一个开箱即用的React Table 组件，支持纯键盘操作、多级表头、排序、合计列、固定列、行拖动、列拖动、等高级功能。

## ✨ 特性

- 🌈 提炼自企业级中后台产品的交互语言和视觉风格。
- 📦 开箱即用的高质量 React 组件。
- 🛡 使用 TypeScript 开发，提供完整的类型定义文件。
- ⚙️ 应用开发框架和设计工具配套。
- 🌍 
- 🎨 基于 CSS-in-JS 的主题定制能力。
